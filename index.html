Why am I seeing this page?

You were getting "CORS" and "Network" errors because you were opening website files directly from your computer (using file:///...). A web server (like Apache in XAMPP) is required to run the website and its PHP backend correctly. By using the links on this page, you ensure everything runs through the server.
